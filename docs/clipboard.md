## Clipboard  
  
#### Copying text from the OS's clipboard  
```python  
FX.Clipboard.CopyFrom() | run     # no args required  
```  
  
#### Copying text to the OS's clipboard  
```python  
FX.Clipboard.CopyTo(  
	content='hello world!'  
) | run  
```
